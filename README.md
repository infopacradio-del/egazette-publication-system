# Gazette Publication Management System

## Overview
A system for publishing government gazette notices with payment handling,
workflow tracking, and departmental approvals.

## Key Features
- Cash and Account clients
- Notice templates (e.g. J193)
- Payment tracking (card & direct deposit)
- POP validation (no duplicate usage)
- Departmental workflow (Verification, Proofreading, DTP, Finance)
- Gazette compilation by province

## Tech Stack
- Frontend: React
- Backend: Node.js (Express)
- Database: PostgreSQL
- Auth: JWT
